# Dijital Güvenlik ve Giriş Sorunları (SOP-SEC-003)

## 1. Mobil Şube Bloke Kaldırma
Müşteri şifresini 3 kez yanlış girerse güvenlik blokesi oluşur.
- **Çözüm 1 (Mobil):** "Şifremi Unuttum" > Kart Bilgileri veya NFC ile kimlik doğrulama.
- **Çözüm 2 (Çağrı Merkezi):** Sesli yanıt sistemi üzerinden.
- **Çözüm 3 (Şube):** Kimlik ibrazı ile.

## 2. Şüpheli İşlem Bildirimi (Fraud)
Müşteri "hesabımdan para çekiliyor" veya "bilmediğim SMS geliyor" derse:
1. **Acil Eylem:** Tüm dijital kanallar (Mobil/Web) geçici olarak kapatılır.
2. Kartlar "Fraud Şüphesi" ile bloke edilir.
3. Harcama kalemleri tek tek teyit edilir.
4. Müşteriye yeni cihaz/SIM değişikliği yapıp yapmadığı sorulur (SIM Swapping riski).

## 3. Cihaz Eşleşmesi Kaldırma
Müşteri telefonunu değiştirdiyse yeni cihaza aktivasyon gerekir.
- Eski cihaz sistemde kayıtlıysa, yeni cihazda "Cihaz Eşleşmesini Kaldır" adımı çıkar.
- Bu adımda SMS onayı ve kart şifresi istenir.

## 4. Sıkça Sorulan Sorular (SSS)
**Soru:** Şifremi unuttum, SMS gelmiyor, ne yapmalıyım?
**Cevap:** SIM kart değişikliği yaptıysanız "SIM Bloke" oluşmuş olabilir. Çağrı merkezini arayarak veya ATM'den bloke kaldırmanız gerekir.

**Soru:** Hesabımdan bilgim dışında para çekildi, ne yapmalıyım?
**Cevap:** Derhal mobil uygulamadan kartınızı geçici olarak kapatın ve 0850'li numaramızdan "Kayıp/Çalıntı" hattına bağlanın.

**Soru:** Mobil şube "Cihaz Eşleşmesi Hatası" veriyor.
**Cevap:** Eski telefonunuz sistemde hala kayıtlı olabilir. "Cihaz Eşleşmesini Kaldır" menüsünden eski kaydı silip tekrar deneyin.
